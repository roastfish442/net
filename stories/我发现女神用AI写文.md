冬枫又更新了。

&nbsp;

黎亦扬猛地把页面拉到最下面，看到点赞数先是一喜，随后看到发布时间，心里又是一沉：才发了三分钟就有六个点赞，多半是在群里事先宣传过了。她退出又点进来，页面下又多了一条评论，写满了夸张的溢美之词。

有什么了不起的，只不过又是那个小团体在抱团取暖，没了她的那些小粉丝，冬枫什么都不是。黎亦扬恨恨地咬了咬牙，突然看到右下角多出新的消息红点，立刻点了进去。

是一条点赞通知，却不是献给她的任何一篇同人，而是一条评论。眼看评论的点赞数比她的任何一篇同人都要高，黎亦扬愤怒地把手机扔到了一边。

人和人的差距为什么这么大？凭什么同样都是同人女，冬枫就如此受人追捧，她却无人问津？黎亦扬胸口发闷，又把手机捡回来，切换到吐黑泥专用的小号，发了一条抱怨的动态：

“某女明星写的也不过如此，真不知道为什么那么多人爱捧她的臭脚。”

&nbsp;

&nbsp;

&nbsp;

动态很快有了几个点赞，黎亦扬舒服多了，又看到大号上有了新消息：

&nbsp;

【小北】：啊啊啊啊你看冬枫太太的新文了吗？太太太好吃了！

【小北】：这是我吃过最好的原作向，又还原又甜，没想到还能那么解读！

&nbsp;

黎亦扬的牙都快咬碎了，但还是耐着性子回复：“是吗？我还没来得及看，之后抽空看看。”

她才不会看，现在不会，以后更不会！

为了避免小北继续赞美冬枫，黎亦扬随手转发了一个搞笑视频给她，以此换了话题。自从退了那个云淡风清同好群，小北是她圈子里为数不多的朋友了，因此就算小北崇拜冬枫，黎亦扬也仍然维护着与她的关系。

把手机放在一边，黎亦扬准备睡觉，心里却总是觉得不舒服。当她意识到自己又点开tag的时候，一切都已经晚了。

冬枫的文笔一向扎实，寥寥数笔就将人带进那个下着大雪的夜晚。上官云淡将皮毛大氅披在季风清肩头，默然无语，两人指尖轻触，有种无声的暧昧。

&nbsp;

“你不要管那些人怎么说。无论如何，你在我心中始终如一。”上官云淡开口道。

&nbsp;

季风清默默无言。他曾是家境优渥的小少爷，但随后家道中落，他饱受冷落和白眼。欺负他的人放肆地叫嚣：“现在季家倒了，还有谁会护着你，宠着你？”

而上官云淡与这些趋炎附势之人都不同，这也是两人关系升温的重要节点。

两人恰到好处的暧昧氛围一直持续到文末，直到被突如其来的不速之客打断——欲知后事如何，请听下回分解。

&nbsp;

呵，也不过如此嘛！黎亦扬读完，又忍不住去看文章下面的评论，果然都是那些熟面孔。真不知道对着这么平庸的文，她们到底是怎么夸出来的，黎亦扬翻了个白眼，准备睡觉，可是刚刚的文章内容在脑海里挥之不去。大雪，廊下，指尖相触，无言的暧昧氛围……为什么这样的文字不是她写出来的？为什么冬枫就能信手拈来似的挥洒她的才能？

黎亦扬气得又拿起手机发了一条动态：

为什么有人的成功来得那么轻松？早晚遭报应！

发完这条，她才松了一口气似的，翻身睡了。

&nbsp;

&nbsp;

第二天下午没课，黎亦扬吃完午饭，一边在学校里散步，一边构思剧情。她打算写一篇云淡风清的娱乐圈pa，影帝x糊咖的日常向小甜饼。光是想象两人相处的各种情节，黎亦扬的脸上就忍不住露出笑容。

只是脑补容易落笔难，等她磕磕绊绊写完这一篇，时间已经到了深夜。看着自己的大作，黎亦扬非常满意：这次的热度肯定很高！就算没有一百，也会有五十吧！

她在期待中入睡，醒来之后第一时间查看消息通知，和往常一样冷冷清清，只有个位数的点赞和一条来自小北的评论。

可能是发得太晚了，没什么人看吧。黎亦扬自我安慰，又收到小北发的消息：“梨子写得真是太甜了，我含泪吃三大碗。”

黎亦扬并没感到有多开心。小北会夸赞她的文，只不过是为了维持她们之间的关系，而不是因为她写的文有多好多有趣。

她收拾心情，下床准备去上课。上午的课无聊透顶，于是她又开始刷云淡风清的tag。刷着刷着，她的心里猛地“咯噔”了一下：冬枫又更新了。

真晦气！黎亦扬顿时没了刷tag的心情。冬枫写了什么她没兴趣，反正少吃她的一口饭也不会掉块肉！

不过小北不知道她心里的这些弯弯绕绕，新消息如期而至：

【小北】：女神又更新了呜呜呜……女神的饭好好吃……

【小北】：可恶，明明冬枫老师勤勤恳恳做饭，质量还那么高，为什么还有人在背后蛐蛐女神

【小北】：心疼老师……

&nbsp;

黎亦扬呼吸一滞。小北这话是什么意思？她知道自己在背后骂冬枫的事，在暗示自己吗？不对，小号的空间只开了好友可见，那个号里根本没有圈内人，小北没可能知道。

&nbsp;

【梨子】：怎么回事？

【小北】：你去看女神最新的文就知道了

&nbsp;

黎亦扬只好打开冬枫的新文看了起来。剧情进展到比武大会，季风清一鸣惊人，让先前瞧不起他的弟子们丑态百出，此后又与上官云淡在比武台上明撕暗秀，让同人女大饱眼福。

黎亦扬快速扫过这些情节，终于找到冬枫在文末的写作感言。

&nbsp;

“终于写到这个情节了，我们小季也终于扬眉吐气啦～其实我也多多少少有和小季相同的经历，直到现在都一直有人在背后攻击我和我的作品。对此我的态度和小季一样，就是堂堂正正，用实力说话。至于那些阴沟里的老鼠，就让他们见鬼去吧！

“感谢小季，是他让我拥有了面对流言蜚语的勇气。”

&nbsp;

&nbsp;

&nbsp;

她知道了？

黎亦扬一阵心惊，立刻自我安慰：没关系，冬枫怎么可能知道她的小号？肯定也有别人在背地里骂她。但黎亦扬隐隐觉得不对，赶快又重读了一遍文章。

&nbsp;

“季风清那点花花架子，连我都不如，也不知师父为什么对他另眼相待。”

“不就是仗着季家以前的那点威风，给他留了点情面吗？没有季家，他根本什么都不是。现在季家倒了，他这个含着金汤匙出生，不知人间疾苦的小少爷，也该吃点苦受点罪了！”

&nbsp;

黎亦扬越看越觉得不对劲。也许是她想得太多，但冬枫文章里的这些流言蜚语，怎么这么像她这几天发的黑泥动态？

说季风清实力不足，全靠家族余威才有今天的地位，说季风清含着金汤匙出生，早该吃苦受罪……不就是她骂冬枫“能力平平，全凭小团体追捧，成功来得太过轻易”的那些话？

肯定是她想多了吧。反派人物不都是这么写的吗？想到这里黎亦扬又愣住了：她也和文里的这些人一样，是反派角色？

黎亦扬立刻气不打一处来了：冬枫这种人懂什么！明明自己的更新频率不下于她，每次写文也都尽心尽力，可是热度连冬枫的零头都比不上！她只不过是不屑于和小团体虚与委蛇，抱团取暖而已！凭什么她就要被当成反派，在冬枫的文里被打得涕泗横流，连连求饶？

她突然灵光一闪：冬枫写文的时候，该不会是把她自己代入了季风清吧？其实她根本就是上官云淡的梦女，披了季风清的皮来幻想与上官云淡的旖旎爱情吧？

黎亦扬的脸上浮现一抹冷笑，她又掌握了冬枫的一项罪证。



&nbsp;

&nbsp;

【小北】：可是写文的时候用自己的经历作为素材也是很正常的事呀，我不觉得这样有什么不好的

【小北】：梨子你不喜欢冬枫老师的话，我以后就不在你面前提了

【小北】：［抱抱］［抱抱］

&nbsp;

算了，小北是个老好人，从来没见她和谁红过脸。黎亦扬不指望她能和自己一起讨厌冬枫，不过小北说会少提起冬枫，这倒是正合黎亦扬的心意。

她又看了一眼自己文章的热度，终于勉强突破两位数了，可冬枫的文章光是红心就已经破百，更不要说数不清的评论和推荐。

&nbsp;

黎亦扬又愤恨地在小号吐露心声：

在文章里卖惨博同情这招真好用啊，全世界的腿毛都吻上来了，恶心不恶心啊。

她没想到，隔天冬枫的更新里，又一次出现了含义相似的句子。

&nbsp;

“只不过是在比武大会上出了点风头，那些弟子就一窝蜂地来巴结他，真是恶心。”

“不过是看他身世凄惨，同情他罢了。”

如果之前的算是巧合，那连番的巧合算什么？黎亦扬身体微微发抖：冬枫知道！她一直在看！

她哆嗦着切换到了小号，一条一条地删除自己发过的动态。不对，比起动态，她更应该删好友！不不不，这个号不能再用了，绝对不能！

黎亦扬又想到了其他可能：冬枫知道这个小号的背后是自己吗？如果冬枫截图挂人，舍弃小号没什么大不了的，可是如果她知道这一切都是自己做的呢？

这一夜，她整晚都在做噩梦，梦见自己身败名裂，成百上千个冬枫的腿毛私信她，用千奇百怪的词语辱骂她，那些文字组成一道鲜红的河流，想将她吞没，又想将她撕裂。

也许梦都是反的，黎亦扬的生活反而风平浪静。她仍旧按部就班地上课，下课，冬枫也按部就班地更新。直到连载结束，冬枫也没有任何要追究那个小号的意思，但完结篇的热度却突破了500大关，评论区上下一片喜气洋洋，恭喜完结，黎亦扬恨得咬牙切齿。

&nbsp;

披皮梦能不能去死啊。df你什么时候去死啊。

手指轻轻一点，满是恨意的文本就这样发送出去。

她又注册了新的小号，这次她一个好友也没加。

&nbsp;

小北发消息问黎亦扬，最近在忙什么？好久没看你写文了，上次那篇没完结的，还等着看更新呢！

黎亦扬只好回复说自己最近卡文写不出来。既然写出来也没人在意，还有写下去的必要吗？自己这样的写手，只能通过抱团取暖才能获得一点热度，可她才不愿意像冬枫那样，通过社交来换取关注！

&nbsp;

【小北】：这样啊……没事，卡文就像便秘，总能拉出来的！

【小北】：梨子加油加油！

&nbsp;

小北是在暗示她写的都是屎吗？

黎亦扬一肚子火不知道往哪里发，又不好拉黑小北，只好把消息删了眼不见为净。

&nbsp;

虽然这阵子她无心写文，但tag没少刷。她看到冬枫又开了新文，评论区里还是那么热闹，又看到冬枫在动态里表达对AI写文的担忧，下面仍然是一片响应。

呵，嘴上说着抵制，私下没少用吧。

黎亦扬总是喜欢恶毒地揣测冬枫的每一句话，仿佛这样能让她解恨似的。她忍不住点开冬枫的新文，打算从里面寻找一些使用AI的蛛丝马迹。

出乎意料的是，还真被她找到了。冬枫的文风变化很大，据说是打算做些新的尝试，增加了文章中的描写部分，使得文章更加偏向华丽意识流，但黎亦扬越看越激动：文章里频繁出现的数字，还有几处AI常用的句式，没有错，冬枫的这篇文是用AI写的！

她几乎急不可耐地打算宣告这一发现，但想到那些“女明星的小腿毛”又犯了难：这群人绝对会无脑护主，就算自己揭穿冬枫用AI写作，又能怎么样？

思来想去，她决定先去找小北谈谈。小北一直以来都很公正，虽然她喜欢冬枫，但如果冬枫也用了AI写作，小北断然不会容忍。

&nbsp;

【小北】：嗯……我觉得这些证据也不是太充足。因为AI写作的特征必然也是人类写作的特征，我们不能太早下结论

【梨子】：［图片］

【梨子】：我去做了鉴定，显示是100%AI生成

【小北】：这种鉴定的结果也不一定准确啊，梨子你先冷静一下，这对冬枫老师是很严重的指控，我们不能那么轻率

【梨子】：那我们还能怎么办？不能让冬枫继续破坏圈子环境了！她一个搞小团体的披皮梦，早就把圈子氛围搞坏了，现在又用AI写文，难道还能容忍她继续这么下去？

【小北】：……

【小北】：梨子，你冷静点

【小北】：我一直都想跟你说，只是找不到合适的时机……再这样嫉妒下去，最后受伤害的还是你自己

&nbsp;

嫉妒？对，她是嫉妒冬枫！嫉妒她拥有自己得不到的热度，嫉妒她在圈里呼风唤雨，只手遮天，嫉妒她写得一手好文章，甚至嫉妒她的宽宏大量——她怎么能就这样不追究？

手机上，小北一条接一条地发来消息：

你说你写的文章没人看，可是我每一篇都在看啊？我喜欢你笔下的云淡风清，他们是很可爱，很甜蜜的一对，你在写下这些情节的时候，一定也满怀着对他们的爱吧！

明明是因为爱才开始写同人，为什么要恨与你一起爱着他们的人呢？我也会嫉妒，我也会心理不平衡，但是梨子，那些你以为他人唾手可得的东西，可能付出了比你想象中多得多的努力和代价……

……

&nbsp;

黎亦扬不知道怎么回复小北，只是一味抱着手机哭泣。她知道她不该这么嫉妒，知道自己已经成了所谓的反派角色，可是如果不恨冬枫的话，如果这一切不是冬枫的错的话……

她就得开始恨自己了。

&nbsp;

指控冬枫用AI写文的计划最终没能实施。黎亦扬删号退圈，在小北的建议下卸载了App，彻底远离了冬枫。

小北怕她寂寞，还会发些搞笑视频来给她解闷。这让她觉得在云淡风清圈里发生的不都是坏事，至少还有小北这样一个朋友。

&nbsp;

转眼一个月过去，黎亦扬已经彻底习惯了不看同人的生活，也再没有在小号里发泄过愤怒，因此，事件真正发生的时候，她几乎毫无准备。

她的QQ收到了大量的好友申请，申请理由有一半都是对她的人身攻击。她正一头雾水的时候，小北的消息也随之而来：

&nbsp;

【小北】：［转发消息］

【小北】：这是你做的吗？

【小北】：如果是你做的，我想我们还是不要做朋友了

&nbsp;

黎亦扬一头雾水地点开链接，面色顿时严肃起来。这是一篇澄清文章，起因是冬枫发布的新文因为不正常的热度增长被质疑买赞，冬枫在澄清自己没有买赞的同时，也给出了她推测的始作俑者：【梨子洋】——这正是黎亦扬的id。冬枫认为，【梨子洋】为自己的文章买赞，就是想栽赃陷害冬枫，给她扣一顶“为热度不择手段”的帽子。

&nbsp;

这不是她做的！黎亦扬大呼冤枉，却在看到下一张图的时候失声。也许是贯彻君子报仇十年不晚的精神，冬枫将她在小号里的所有动态逐一列出：她骂冬枫是女明星，搞小团体，卖惨，养腿毛，披皮梦，诅咒她身败名裂，滚出云淡风清，早点去死……

这一小号和指责冬枫买赞的账号ip地址正好在同个省份，究竟是巧合还是必然？至少浏览澄清条的人们心中都有答案。

&nbsp;

澄清文章的最后，冬枫用加粗字体如此询问：

我和你无冤无仇，为什么那么恨我呢？

&nbsp;

黎亦扬几乎百口莫辩。她已经不恨了，或者说，没那么恨了！可是说过的话做过的事不能撤回，这就是她为自己的嫉妒应当付出的代价。

她赶快回复小北，说这不是自己做的，自己连APP都卸载了，怎么会给冬枫买赞？但小北已经把她拉黑了。

&nbsp;

黎亦扬几乎崩溃。她早就没那么在乎什么圈内名声，可唯独失去小北这个朋友，是让她最难过的事。她想澄清自己没有买赞，但评论区群情激愤，她根本不敢留言，只好点开冬枫的私信聊天窗口，言辞恳切地写了很长一段话。

冬枫老师你好，我对之前的所作所为感到很抱歉。因为嫉妒你的热度和人气，我说了很多过分的话，真的很对不起。
